# Number Guessing Game

A simple, interactive web-based game where the player tries to guess a randomly generated number between 1 and 100.

## Features
- **Real-time Hints:** Informs the player if their guess is too high or too low.
- **Score Tracking:** Displays the total number of attempts upon winning.
- **Input Validation:** Error handling for empty or invalid inputs.
- **Responsive Design:** Clean, centered UI that works on different screen sizes.

## Technologies Used
- **HTML5:** For the structure of the game.
- **CSS3:** For custom styling and layout.
- **JavaScript (ES6+):** For the game logic, random number generation, and DOM manipulation.

## How to Play
1. Enter a number between **1 and 100** in the input field.
2. Click the **Submit** button.
3. Read the feedback message to adjust your next guess.
4. Keep guessing until you find the correct number!
